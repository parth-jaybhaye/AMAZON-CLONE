# 🛒 Amazon Homepage Clone

A responsive Amazon Homepage Clone built using **HTML** and **CSS** as part of my Web Development learning journey.

This project helped me understand how real-world websites are structured and gave me hands-on practice with building responsive layouts from scratch.

---

## ✨ Features

- 🏠 Amazon-inspired homepage layout
- 🧭 Responsive navigation bar
- 🔍 Search bar UI
- 📦 Product category cards
- 🖼️ Hero banner section
- 📑 Footer with multiple sections
- 📱 Responsive design using CSS
- 🎨 Clean UI built with HTML & CSS only

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- Media Queries

- Building a real-world website clone
